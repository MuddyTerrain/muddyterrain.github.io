**Welcome to the official documentation for GenAI Model Generator, a powerful Unreal Engine plugin for AI-powered 3D model generation and PBR texture creation.** This plugin connects your projects to leading AI 3D generation services — Meshy, Hyper3D Rodin, Tripo AI, fal.ai, and Google — through a single, unified interface.

## What Can You Do?

- **Text-to-3D** — Describe a model in plain text and generate a 3D asset
- **Image-to-3D** — Convert concept art or photos into 3D models
- **Retexturing** — Apply new AI-generated textures to existing meshes
- **PBR Texture Generation** — Create base color, normal, roughness, and metallic maps with AI
- **Multiple Formats** — Export as GLB, FBX, OBJ, USDZ, or STL
