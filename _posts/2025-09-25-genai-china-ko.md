<p><strong>중국 최고의 AI 모델을 언리얼 엔진 프로젝트에서 활용하세요.</strong> Gen AI China는 Alibaba, Bytedance, Moonshot AI 등의 프로바이더가 제공하는 최첨단 생성형 AI를 위한 안정적이고 통합된 강력한 인터페이스를 제공하는 프로덕션 레디 플러그인입니다. 여러분은 게임플레이에만 집중하세요. 통합 레이어는 저희가 처리합니다.</p>

  <h2 style="margin-top: 0;">두 개의 리전, 하나의 플러그인!</h2>
  <p>Alibaba Cloud의 이중 API 구조를 완벽하게 지원합니다. 플러그인이 <strong>중국 본토</strong>와 <strong>싱가포르 (전세계)</strong> 리전에 맞는 올바른 엔드포인트로 지능적으로 요청을 라우팅하고, 각 리전에 맞는 API 키를 사용하여 플레이어의 위치와 관계없이 원활한 통합을 보장합니다.</p>
  <p>이 플러그인은 새로운 모델과 기능이 출시될 때 <strong>무료 업데이트</strong>를 제공합니다. 유용하게 사용하고 계신다면, Fab에서 <strong>별 5개 평가</strong>로 응원해 주세요!</p>
</div>

<h2>주요 기능:</h2>
<ul>
    <li>
        <p><strong>독점 모델 접근:</strong> 🚀<br>
        Alibaba의 Qwen 시리즈, Bytedance의 Skylark & DeepSeek, Moonshot AI의 Kimi 모델 등 중국 최고의 AI 프로바이더가 제공하는 강력한 모델들을 하나의 통합된 API를 통해 사용하세요.</p>
    </li>
    <li>
        <p><strong>실시간 텍스트 스트리밍:</strong> ⚡️<br>
        역동적인 실시간 응답으로 캐릭터에 생명을 불어넣으세요. 스트리밍 지원을 통해 텍스트가 생성되는 즉시 청크 단위로 전달되어, 실시간 내레이션 대사와 반응형 AI 동료를 대기 시간 없이 구현하기에 완벽합니다.</p>
    </li>
    <li>
        <p><strong>고급 채팅 완성:</strong> 🤖<br>
        다양한 고급 언어 모델을 활용하여 동적 NPC 대화, 절차적 퀘스트 생성, 실시간 내러티브 적응으로 게임을 강화하세요.</p>
    </li>
    <li>
        <p><strong>멀티모달 비전/텍스트 입력:</strong> 📸<br>
        모델이 게임을 '볼' 수 있게 하세요. 스크린샷이나 기타 이미지를 텍스트와 함께 직접 입력으로 제공하여, AI 기반 관전 해설이나 환경 인식 NPC 상호작용을 구현하기에 완벽합니다.</p>
    </li>
    <li>
        <p><strong>고품질 이미지 생성:</strong> 🖼️<br>
        Alibaba의 Qwen-Image, Bytedance의 Seedream 등의 모델을 지원하여 최첨단 이미지 생성을 워크플로우에 직접 통합하세요.</p>
    </li>
    <li>
        <p><strong>텍스트 음성 변환 (TTS):</strong> 🔊<br>
        Alibaba의 Qwen-TTS 등의 모델을 사용하여 인게임 대화와 내레이션을 위한 역동적이고 고품질의 음성을 생성하세요.</p>
    </li>
    <li>
        <p><strong>안전하고 중앙화된 인증:</strong> 🔐<br>
        하나의 안전한 중앙 설정 패널에서 모든 API 키를 관리하세요. 이동 불가능한 암호화 파일을 사용하여 API 키를 소스 컨트롤과 분리하여 안전하게 보관합니다.</p>
    </li>
    <li>
        <p><strong>Blueprint & C++ 지원:</strong> 🔗<br>
        빠른 반복 작업을 위한 간단하고 강력한 Blueprint 노드를 활용하고, C++에서도 동일한 견고한 자기 관리형 함수에 접근하여 최대 성능과 제어를 달성하세요.</p>
    </li>
     <li>
        <p><strong>크로스 플랫폼 설계:</strong> 🌐<br>
        플러그인은 네트워킹과 파일 관리를 위해 표준적인 고수준 언리얼 엔진 모듈만을 사용하므로, 엔진이 지원하는 모든 플랫폼에서 동작합니다.</p>
    </li>
</ul>

<h2>활용 사례:</h2>
<ul>
    <li><strong>진정한 동적 NPC:</strong> 플레이어의 행동과 월드 이벤트를 기반으로 특정 문화적 맥락에 맞게 실시간으로 대화가 생성되는 캐릭터를 만드세요.</li>
    <li><strong>절차적 콘텐츠 생성:</strong> 중국 역사와 신화에서 영감을 받은 퀘스트와 아이템 설명부터 방대한 설정집까지, 풍부하고 일관된 콘텐츠를 런타임에 즉시 생성하세요.</li>
    <li><strong>인터랙티브 음성 기반 게임플레이:</strong> TTS와 언리얼의 음성 입력을 결합하여 인게임 캐릭터와 실제 음성 대화를 나누세요.</li>
    <li><strong>빠른 프로토타이핑 & 월드 빌딩:</strong> 에디터에서 직접 이미지 생성을 활용하여 컨셉 아트와 방대한 월드 데이터를 빠르게 생성하세요.</li>
</ul>

<h2>이 플러그인을 선택해야 하는 이유</h2>
<ul>
    <li><strong>끊임없는 진화:</strong> 🧬 중국 주요 AI 기업들의 새로운 모델과 기능이 출시되는 즉시 신속하게 업데이트를 제공합니다.</li>
    <li><strong>프로덕션 레디 & 검증 완료:</strong> ⚔️ 모든 기능은 실제 API 키를 사용하여 다양한 사용 사례와 규모에서 정기적으로 테스트됩니다.</li>
    <li><strong>범용적이고 유연한 설계:</strong> 🌵 깔끔하고 범용적인 접근 방식으로 구축되어 미래의 모델 및 API 변경에 빠르게 적응할 수 있습니다.</li>
    <li><strong>개발자에 의한, 개발자를 위한 제품:</strong> 💻 수년간의 전문 게임 개발 경험을 바탕으로 안정적이고 잘 문서화된 고성능 구현을 보장합니다.</li>
    <li><strong>전담 지원:</strong> 📧 질문이나 커스텀 수정 요청은 지원팀에 이메일로 문의하세요. 기꺼이 도와드리겠습니다.</li>
</ul>
