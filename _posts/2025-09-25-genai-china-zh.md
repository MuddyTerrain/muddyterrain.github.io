<h2>核心功能：</h2>
<ul>
    <li>
        <p><strong>独家模型接入：</strong> 🚀<br>
        通过单一统一的 API 集成来自中国顶级 AI 提供商的强大模型，包括阿里巴巴的 Qwen 系列、字节跳动的 Skylark 和 DeepSeek，以及 Moonshot AI 的 Kimi 模型。</p>
    </li>
    <li>
        <p><strong>实时文本流式传输：</strong> ⚡️<br>
        通过动态的实时响应让您的角色栩栩如生。我们的流式传输支持在文本生成的同时分块推送，非常适合创建实时旁白对话和无需等待的响应式 AI 伙伴。</p>
    </li>
    <li>
        <p><strong>高级聊天补全：</strong> 🤖<br>
        使用丰富的高级语言模型，为您的游戏提供动态 NPC 对话、程序化任务生成和实时叙事适配。</p>
    </li>
    <li>
        <p><strong>多模态视觉/文本输入：</strong> 📸<br>
        让模型能够"看见"您的游戏。将截图或其他图像与文本一起作为直接输入——非常适合创建 AI 驱动的观战解说或环境感知的 NPC 交互。</p>
    </li>
    <li>
        <p><strong>高质量图像生成：</strong> 🖼️<br>
        将最先进的图像生成直接集成到您的工作流程中，支持阿里巴巴的 Qwen-Image 和字节跳动的 Seedream 等模型。</p>
    </li>
    <li>
        <p><strong>文本转语音（TTS）：</strong> 🔊<br>
        使用阿里巴巴的 Qwen-TTS 等模型，为游戏内对话和旁白生成动态、高品质的语音。</p>
    </li>
    <li>
        <p><strong>安全且集中的身份验证：</strong> 🔐<br>
        在一个安全、集中的设置面板中管理所有 API 密钥。我们的系统使用不可移植的加密文件来保护您的密钥安全，并与源代码控制隔离。</p>
    </li>
    <li>
        <p><strong>Blueprint 和 C++ 双重支持：</strong> 🔗<br>
        享受简洁、强大的 Blueprint 节点实现快速迭代，同时在 C++ 中访问同样健壮的自管理函数，获得最佳性能和控制力。</p>
    </li>
     <li>
        <p><strong>为跨平台而设计：</strong> 🌐<br>
        该插件仅使用标准的高级虚幻引擎网络和文件管理模块，确保在引擎支持的所有平台上都能正常运行。</p>
    </li>
</ul>

<h2>应用场景：</h2>
<ul>
    <li><strong>真正的动态 NPC：</strong>创建能够根据玩家行为和世界事件实时生成对话的角色，并针对特定文化背景进行定制。</li>
    <li><strong>程序化内容生成：</strong>在运行时即时生成丰富、连贯的内容，从任务和物品描述到以中国历史和神话为灵感的完整传说典籍。</li>
    <li><strong>互动语音驱动的游戏玩法：</strong>将 TTS 与虚幻引擎的语音输入相结合，与游戏内角色进行真实的语音对话。</li>
    <li><strong>快速原型制作与世界构建：</strong>在编辑器中直接使用图像生成功能，快速创建概念艺术和大量世界数据。</li>
</ul>
